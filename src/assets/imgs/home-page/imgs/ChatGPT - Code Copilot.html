def merge(a, b):

if a == []: return b

if b == []: return a

if a[0] < b[0]:

return [a[0]] + merge(a[1:], b)

else:

return [b[0]] + merge(a, b[1:])


It is a common function, such as you know from MergeSort, for example. It has two lists on input [a0,..., an-1] and [bo,..., bm-1]. If one of them is empty, the result is the second list. If both a and a and b are non-empty, the result is a list with a0 first, followed by the recursively merged lists [a1,...,an-1] and [b0,...,bm-1]. If b0 <a0, the result is analogous.

If we have lists [a0,..., an-1] and [b0,..., bn-1] that are sorted, calling merge(a,b) will return the sorted result. However, we can also call the merge function in a situation where the input lists are not sorted, e.g. merge ([3,2,7,1], [6,4,8,5]) returns [3,2,6,4,7,1,8,5].
